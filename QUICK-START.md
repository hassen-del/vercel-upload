# ⚡ QUICK START - 5 MINUTE GUIDE

## 🎯 YOUR GOAL:
Get client portal live at: `client.tradeleadsflow.com`

---

## ✅ IN 5 STEPS:

```
┌─────────────────────────────────────────┐
│  STEP 1: Zip the Folder                │
│  📦 Right-click → Compress              │
│     Time: 10 seconds                    │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│  STEP 2: Go to Vercel                  │
│  🌐 vercel.com/new                      │
│     Time: 5 seconds                     │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│  STEP 3: Upload & Deploy               │
│  ⬆️  Drag ZIP → Click Deploy            │
│     Time: 3 minutes (wait for build)    │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│  STEP 4: Add Domain                    │
│  🔗 Settings → Domains → Add            │
│     Type: client.tradeleadsflow.com     │
│     Time: 30 seconds                    │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│  STEP 5: Update DNS                    │
│  📝 CNAME: client → vercel              │
│     Time: 1 minute + 10 min wait        │
└─────────────────────────────────────────┘
                  ↓
              🎉 DONE!
```

---

## 📋 COPY-PASTE CHECKLIST:

```
□ Compress vercel-upload folder
□ Go to vercel.com/new
□ Upload the ZIP file
□ Click Deploy button
□ Wait for green checkmark
□ Test .vercel.app URL
□ Settings → Domains
□ Add: client.tradeleadsflow.com
□ Copy DNS instructions
□ Update DNS (CNAME record)
□ Wait 10-30 minutes
□ Test: client.tradeleadsflow.com
□ Share with client!
```

---

## 🔧 DNS RECORD (COPY THIS):

```
Type:  CNAME
Name:  client
Value: cname.vercel-dns.com
TTL:   Auto or 600
```

**Add this to wherever you manage tradeleadsflow.com DNS**

---

## 🎯 FINAL URL:

```
https://client.tradeleadsflow.com/client
```
