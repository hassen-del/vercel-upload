## 🧪 How to Run the Test

### 5-Minute Setup:

**Step 1: Create Supabase Table** (2 min)
1. Go to: https://supabase.com
2. SQL Editor → New Query
3. Paste SQL from SUPABASE_SETUP_GUIDE.md
4. Click: Run

**Step 2: Create Test Data** (2 min)
1. Open: https://vercel-upload-psi.vercel.app/client
2. Press: F12 → Console
3. Paste: CONSOLE_SETUP_SCRIPT.js
4. Run: `setupAndCreateTestData()`

**Step 3: Verify** (1 min)
1. Wait 30 seconds
2. See metrics in dashboard
3. Verify all 4 tabs work
